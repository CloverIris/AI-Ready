# AI-Ready 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品愿景
AI-Ready 是一款面向个人用户和云算力使用者的 AI 部署管理工具，解决本地 AI 环境配置复杂和云服务器使用门槛高的问题，让每个人都能轻松运行开源 AI 项目。

### 1.2 核心价值主张
- **零门槛本地部署**：像安装普通软件一样部署 Stable Diffusion、Ollama 等 AI 项目
- **一站式云算力管理**：通过桌面端轻松管理远程 Linux 服务器上的 AI 实例
- **环境可迁移**："末影箱"机制实现环境备份与跨服务器迁移

### 1.3 目标用户
| 用户类型 | 特征 | 核心需求 |
|---------|------|---------|
| AI 爱好者 | 有 RTX 显卡，非技术背景 | 一键安装，开箱即用 |
| 创意工作者 | 设计师/内容创作者 | 快速部署 ComfyUI 等工作流 |
| 云算力用户 | 租用 AutoDL 等平台 GPU | 简化 SSH 与远程管理 |
| 学习者 | 想了解 AI 部署的小白 | 教育引导，概念科普 |

## 2. 功能需求

### 2.1 本地模式 (Windows Desktop)

#### F1: 系统检测与建议
- **F1.1** 硬件检测：自动识别 GPU 型号、显存、CUDA 版本支持
- **F1.2** 存储检测：检查磁盘空间，给出推荐配置
- **F1.3** 兼容性报告：生成"是否适合本地部署 AI"的评估报告

#### F2: 环境自动化配置
- **F2.1** Python SDK 管理：Miniconda 静默安装
- **F2.2** 多 Python 版本管理：支持 3.9/3.10/3.11 等版本并存
- **F2.3** 虚拟环境管理：GUI 化的 Conda 环境增删改查
- **F2.4** 环境变量管理：可视化 PATH 配置，自动避免冲突

#### F3: AI 项目一键安装
- **F3.1** 预设工作流：
  - Stable Diffusion + ComfyUI
  - Ollama + WebUI
  - vLLM + OpenAI 兼容接口
  - 其他热门开源项目
- **F3.2** 依赖自动解决：自动安装 CUDA、cuDNN 等运行时
- **F3.3** 端口管理：自动检测并分配可用端口

#### F4: 教育引导系统
- **F4.1** OOBE 新手引导：Windows 安装向导式体验
- **F4.2** Tips 系统：解释 PATH、CUDA、VENV 等概念
- **F4.3** 内容推送：AI 日报、新开源项目推荐、玩法教程

#### F4.5: AI Skills 生态 (新增)
- **F4.5.1** Skills 商店：浏览、搜索、安装 AI Skills（预配置提示词模板）
- **F4.5.2** Skills 管理：已安装 Skills 的启用/禁用、配置、更新
- **F4.5.3** Skills 收藏：收藏感兴趣的 Skills，方便后续安装
- **F4.5.4** Skills 分类：按编程开发、内容创作、数据分析、系统管理等分类

#### F4.6: MCP 集成 (新增)
- **F4.6.1** MCP 市场：浏览和安装 MCP Servers（Model Context Protocol）
  - 集成官方 MCP Registry API: `https://registry.modelcontextprotocol.io/v0/servers`
  - 支持搜索、分类筛选、分页浏览
  - 显示服务器元数据（描述、版本、传输类型、包信息）
- **F4.6.2** MCP 配置管理：配置本地 MCP 服务器连接参数
- **F4.6.3** MCP 工具发现：自动发现 MCP Servers 提供的 Tools、Resources、Prompts
- **F4.6.4** MCP 运行监控：查看 MCP Server 运行状态和日志
- **F4.6.5** 自定义 MCP：支持添加本地或远程自定义 MCP Servers
- **F4.6.6** 数据缓存：本地缓存 Registry 数据，支持离线浏览

#### F4.7: Skills 数据源集成 (新增)
- **F4.7.1** 官方数据源：从 GitHub `anthropics/skills` 仓库获取 Skills
- **F4.7.2** 静态备用：内置本地 `skills-registry.json` 作为离线备用
- **F4.7.3** 定期同步：每日自动同步最新 Skills 数据
- **F4.7.4** 元数据解析：解析 SKILL.md 文件提取名称、描述、分类、标签

### 2.2 云端模式 (Cloud Management)

#### F5: SSH 连接管理
- **F5.1** 密钥管理：支持密码和 SSH Key 认证
- **F5.2** 连接配置：保存多个云服务器配置
- **F5.3** 自动重连：断线自动恢复机制

#### F6: 服务端 Agent 部署
- **F6.1** 自动上传：通过 SSH 管道上传 Agent 程序
- **F6.2** 远程安装：一键安装 Docker 与 Agent
- **F6.3** 心跳机制：实时获取服务器状态

#### F7: 云端 AI 实例管理
- **F7.1** Docker 容器管理：基于成熟镜像一键部署
- **F7.2** 性能遥测：GPU 利用率、显存、网络监控
- **F7.3** 日志查看：远程容器日志实时查看

#### F8: 末影箱文件系统
- **F8.1** 远程文件浏览：类似 VS Code 的 SFTP 文件管理
- **F8.2** 双向同步：本地与服务器文件同步
- **F8.3** 虚拟磁盘映射（V2）：挂载为 Windows 盘符

#### F9: 网络透传
- **F9.1** SSH 反向隧道：将本地网络代理到服务器
- **F9.2** 解决云厂商联网限制：实现服务器访问受限资源

#### F10: 环境迁移
- **F10.1** 全量备份：导出服务器完整环境
- **F10.2** 清单模式：仅记录配置清单，快速重建
- **F10.3** 跨服务器迁移：支持环境导出到本地再导入新服务器

### 2.3 SaaS 与内容平台

#### F11: 内容运营
- **F11.1** 科普教程：AI 部署基础知识视频/文章
- **F11.2** AI 日报：每日开源项目、AI 新闻推送
- **F11.3** 社区模板：用户分享的工作流模板

#### F12: 商业合作
- **F12.1** 云厂商集成：AutoDL 等平台的一键开通与支付
- **F12.2** 优惠链接：推广返佣与优惠券分发
- **F12.3** 高级功能：跨设备同步等增值服务

## 3. 非功能性需求

### 3.1 性能
- 启动时间 < 3 秒
- SSH 连接建立 < 5 秒
- 文件列表加载 < 2 秒（百级文件）

### 3.2 兼容性
- Windows 10 21H2+ / Windows 11
- 支持 x64 架构
- 支持 NVIDIA GTX 10 系列及以上显卡

### 3.3 安全性
- SSH 密钥本地加密存储 (Windows DPAPI)
- 敏感操作二次确认
- 开源代码审计友好

### 3.4 可维护性
- 模块化架构，核心与 UI 分离
- 完善的日志系统
- 自动更新机制

## 4. 版本规划

### MVP (Phase 1)
- 本地：Miniconda 管理 + Ollama 一键安装
- 云端：SSH 连接 + Docker 部署 Agent + 基础文件管理

### Phase 2
- 末影箱完整功能
- 3-5 个预设工作流
- 内容推送系统

### Phase 3
- Android 移动端
- 云厂商合作集成
- 环境迁移功能
